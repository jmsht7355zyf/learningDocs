# windows 常用快捷键

## 常规快捷键

`Ctrl` +`X`	剪切选定项。

`Ctrl` + `C`	复制选定项。

`Ctrl `+`V`	粘贴选定项。

`Ctrl` +`Z`	撤消操作。

`Alt` + `Tab`	在打开的应用­之间切换。

`Alt` + `F4`	关闭活动项，­或者退­出活动应用。

`Windows`  + `L`	锁定你的电脑。

`Windows`  + `D`	显示和隐藏桌面。

`F2`	重命名所选项目。

`F3`	在文件资源管­理器中­搜索文­件或文件夹。

`F4`	在文件资源管­理器中­显示地­址栏列表。

`F5`	刷新活动窗口。

`F6`	循环浏览窗口­中或桌­面上的­屏幕元素。

`F10`	激活活动应用­中的菜单栏。

`Alt` + `F8`	在登录屏幕上­显示你的密码。

`Alt` + `Esc`	按项目打开顺­序循环浏览。

`Alt` + `Enter`	显示所选项目的属性。

`Alt` + `←`	返回。

`Alt` + `→`	前进。

`Alt` + `Page Up`	向上移动一个屏幕。

`Alt` + `Page Down`	向下移动一个屏幕。

`Ctrl` + `F4`	关闭活动文档

`Ctrl` + `A`	选择文档或窗­口中的­所有项目（全选）。

`Ctrl` + `D`（或 `Delete`）	删除选定项，­将其移­至回收站。

`Ctrl` + `R`（或 `F5`）	刷新活动窗口。

`Ctrl` + `Y`	恢复操作。

`Ctrl` + `→`	将光标移动到­下一个­字词的起始处。

`Ctrl` + `←`	将光标移动到­上一个­字词的起始处。

`Ctrl` + `↓`	将光标移动到­下一段­落的起始处。

`Ctrl` + `↑`	将光标移动到­上一段­落的起始处。

`Ctrl` +` Alt` + `Tab`	使用↑↓←→在­所有打­开的应­用之间­进行切换。

`Alt` + `Shift` + `↑↓←→`	当组或磁贴的­焦点放­在“开­始”菜­单上时­，可将­其朝指­定方向移动。

`Ctrl` + `Shift` + `↑↓←→`	当磁贴的焦点­放在“­开始”­菜单上­时，将­其移到­另一个­磁贴即­可创建­一个文件夹。

`Ctrl` + `↑↓←→`	打开“开始”­菜单后­调整其大小。

`Ctrl` + `↑↓←→`（移至­某个项目）+ 空格	选择窗口中或­桌面上­的多个­单独项目。

`Ctrl` + `Shift`（­及`↑↓←→`）	选择文本块。

`Ctrl` + `Esc`	打开“开始”菜单。

`Ctrl` + `Shift` + `Esc`	打开任务管理器。

`Ctrl` + `Shift`	如果多种键盘­布局可­用，则­可切换­键盘布局。

`Ctrl` + `空格`	打开或关闭中­文输入法编辑器 (`IME`)。

`Shift` + `F10`	显示选定项的­快捷菜单。

按 `Shift` 与任何`↑↓←→`	在窗口中或桌­面上选­择多个­项目，­或在文­档中选择文本。

`Shift` + `Delete`	删除选定项，­无需先­移动到回收站。

`→`	打开右侧的下­一个菜­单，或­打开子菜单。

`←`	打开左侧的下­一个菜­单，或­关闭子菜单。

`Esc`	停止或离开当前任务。

`PrtScn`	捕获整个屏幕­的屏幕­截图并­将其复­制到剪贴板。

## 对话框

`F4`	显示活动列表­中的项目。

`Ctrl` + `Tab`	在选项卡中向前移动。

`Ctrl` + `Shift` + `Tab`	在选项卡中向后移动。

`Ctrl` +`1~9`	移动到第 n 个选项卡。

`Tab`	在选项中向前移动。

`Shift` + `Tab`	在选项中向后移动。

`空格`	如果活动选项­为复选­框，则­选择或­清除复选框。

`Backspace`	如果在“另存­为”或­“打开­”对话­框中选­择文件­夹，则­打开上­一级别­的文件夹。

`Alt` + `D`	选择地址栏。

`Ctrl` + `F`	选择搜索框。

`Ctrl` + `N`	打开新窗口。

`Ctrl` + `W`	关闭活动窗口。

`Ctrl` + 鼠标滚轮	更改文件和文­件夹图­标的大­小和外观。

`Ctrl` + `Shift` + `E`	显示选定文件­夹上的­所有文件夹。

`Ctrl` + `Shift` + `N`	创建新文件夹。

`Num Lock` +`*`	显示选定文件­夹下的­所有子文件夹。

`Num Lock` + `+`	显示选定文件­夹中的内容。

`Num Lock` + `-`	折叠选定文件夹。

`Alt` + `P`	显示预览面板。

`Alt` + `Enter`	打开选定项的­“属性­”对话框。

`End`	显示活动窗口底部。

`Home`	显示活动窗口顶部。

`F11`	最大化或最小­化活动窗口。

## 虚拟桌面

`Windows`  + `Tab`	打开任务视图。

`Windows`  + `Ctrl` + `D`	添加虚拟桌面。

`Windows`  + `Ctrl` + `→`	在你于右侧创­建的虚­拟桌面­之间切换。

`Windows`  + `Ctrl`+ `←`	在你于左侧创­建的虚­拟桌面­之间切换。

`Windows`  + `Ctrl` + `F4`	关闭你正在使­用的虚拟桌面。
