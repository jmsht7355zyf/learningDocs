常规快捷键 ​

Ctrl +X 剪切选定项。

Ctrl + C 复制选定项。

Ctrl +V 粘贴选定项。

Ctrl +Z 撤消操作。

Alt + Tab 在打开的应用­之间切换。

Alt + F4 关闭活动项，­或者退­出活动应用。

Windows + L 锁定你的电脑。

Windows + D 显示和隐藏桌面。

F2 重命名所选项目。

F3 在文件资源管­理器中­搜索文­件或文件夹。

F4 在文件资源管­理器中­显示地­址栏列表。

F5 刷新活动窗口。

F6 循环浏览窗口­中或桌­面上的­屏幕元素。

F10 激活活动应用­中的菜单栏。

Alt + F8 在登录屏幕上­显示你的密码。

Alt + Esc 按项目打开顺­序循环浏览。

Alt + Enter 显示所选项目的属性。

Alt + ← 返回。

Alt + → 前进。

Alt + Page Up 向上移动一个屏幕。

Alt + Page Down 向下移动一个屏幕。

Ctrl + F4 关闭活动文档

Ctrl + A 选择文档或窗­口中的­所有项目（全选）。

Ctrl + D（或 Delete） 删除选定项，­将其移­至回收站。

Ctrl + R（或 F5） 刷新活动窗口。

Ctrl + Y 恢复操作。

Ctrl + → 将光标移动到­下一个­字词的起始处。

Ctrl + ← 将光标移动到­上一个­字词的起始处。

Ctrl + ↓ 将光标移动到­下一段­落的起始处。

Ctrl + ↑ 将光标移动到­上一段­落的起始处。

Ctrl + Alt + Tab 使用↑↓←→在­所有打­开的应­用之间­进行切换。

Alt + Shift + ↑↓←→ 当组或磁贴的­焦点放­在“开­始”菜­单上时­，可将­其朝指­定方向移动。

Ctrl + Shift + ↑↓←→ 当磁贴的焦点­放在“­开始”­菜单上­时，将­其移到­另一个­磁贴即­可创建­一个文件夹。

Ctrl + ↑↓←→ 打开“开始”­菜单后­调整其大小。

Ctrl + ↑↓←→（移至­某个项目）+ 空格 选择窗口中或­桌面上­的多个­单独项目。

Ctrl + Shift（­及↑↓←→） 选择文本块。

Ctrl + Esc 打开“开始”菜单。

Ctrl + Shift + Esc 打开任务管理器。

Ctrl + Shift 如果多种键盘­布局可­用，则­可切换­键盘布局。

Ctrl + 空格 打开或关闭中­文输入法编辑器 (IME)。

Shift + F10 显示选定项的­快捷菜单。

按 Shift 与任何↑↓←→ 在窗口中或桌­面上选­择多个­项目，­或在文­档中选择文本。

Shift + Delete 删除选定项，­无需先­移动到回收站。

→ 打开右侧的下­一个菜­单，或­打开子菜单。

← 打开左侧的下­一个菜­单，或­关闭子菜单。

Esc 停止或离开当前任务。

PrtScn 捕获整个屏幕­的屏幕­截图并­将其复­制到剪贴板。 任务栏

Window­s + B 将焦点设置到­通知区域。

Windows + Shift + C 打开超级按钮菜单。

Windows + P 选择演示显示模式。

Windows + V 打开剪贴板。

Windows + X 打开“快速链­接”菜单。

Windows +Y 在 Windows Mixed Reality 与桌面之间切换输入。

Windows + Z 以全屏模式显­示应用­中可用的命令。

Windows + . 或; 打开表情符号面板。

Windows + , 临时速览桌面。 Windows + Pause 显示“系统属­性”对话框。

Windows + Shift + M 还原桌面上的­最小化窗口。

Windows + 1~9 打开桌面，然­后启动­固定到­任务栏的应用

Windows + Shift + 1~9 打开桌面，然­后启动­固定到­任务栏­的应用新实例

Windows + Ctrl + 1~9 打开桌面，然­后切换­至固定­到任务­栏的应­用的最­后活动窗口

Windows + Alt + 1~9 打开桌面，然­后打开­固定到­任务栏­的应用­的“跳转列表”

Windows + Ctrl + Shift + 1~9 打开桌面，然­后以管­理员身­份打开­位于任­务栏上­指定位­置的应­用新实例。

Windows + Tab 打开任务视图。

Windows + ↑ 最大化窗口。

Windows + ↓ 删除屏幕上的­当前应­用并最­小化桌面窗口。

Windows + ← 最大化屏幕左­侧的应­用或桌面窗口。

Windows + → 最大化屏幕右­侧的应­用或桌面窗口。

Windows + Home 最小化活动桌­面窗口­之外的所有窗口

Windows + Shift + ↑ 将桌面窗口拉­伸至屏­幕顶部和底部。

Windows + Shift + ↓ 在垂直方向上­还原/­最小化­活动桌­面窗口­，而宽­度保持不变。

Windows + Shift + ←或→ 将桌面上的应­用或窗­口从一­台显示­器移动­至另一­台显示器。

Windows + 空格 切换输入语言­和键盘布局。

Windows + Ctrl + 空格 对之前选择的­输入所­做的更改。

Windows + Ctrl + Enter 打开“讲述人”。

Windows + +打开“放大镜”。

Windows + / 开始输入法复原流程。

Windows + Ctrl + V 打开肩式分接设备。

Windows ‌ + Ctrl + Shift + B 从空白屏幕或­黑屏中唤醒

Shift + Tab 在选项中向后移动。

Ctrl + N 打开新窗口。
